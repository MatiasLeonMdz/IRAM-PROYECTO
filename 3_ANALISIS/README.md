# 3_ANALISIS/

## Qué contiene
Objetivo 3 del proyecto: todo lo que es insumo o producto del Análisis A/B y la Diplomatura UTN — diseño del análisis, checklists, auditorías de continuidad, charlas de diseño 7-12, prueba de fuga de memoria, y los entregables finales (portafolio + tarea UTN).

Esta carpeta reemplaza lo que antes eran dos cosas separadas (`3_PORTAFOLIO_UTN` + material suelto de análisis en la raíz) — se fusionaron porque ambos son el mismo Objetivo 3.

- `tarea_UTN/consignas/` — consignas oficiales de la diplomatura (PDF + MD)
