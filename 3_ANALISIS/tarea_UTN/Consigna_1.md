# PARTE 1: Descubriendo el Poder de los Datos
## Consigna de la Entrega 1 (Abarca Módulos 1, 2 y 3)

## 📝 Introducción y Objetivo

¡Bienvenidos a la primera fase de su Proyecto Integrador! En esta etapa, se pondrán el sombrero de Cientistas de Datos. El objetivo es que elijan una situación de la vida real o de una organización, consigan un conjunto de datos (un archivo Excel o CSV con filas y columnas), lo limpien, descubran qué historias nos cuentan esos datos y piensen conceptualmente cómo el Machine Learning podría ayudarlos a resolver el problema identificado.

## 🛠 ¿Qué deben hacer? (Paso a Paso)

### Paso 1: Elegir el problema y los datos (Módulo 1)

- **Definan el contexto:** Elijan un tema o sector que les interese o donde tengan acceso a información (ej. ventas de un comercio local, abandono de clientes en un gimnasio, uso de energía en el hogar, rendimiento de alumnos, control de stock, etc.).
- **Consigan los datos:** El set de datos debe tener forma de tabla (filas y columnas) y contar con un mínimo de 50 filas. Para obtenerlo, pueden:
  - Descargarlo de internet (ej. buscando en repositorios abiertos como Kaggle).
  - Usar datos reales de su entorno laboral o personal (recuerden anonimizar nombres, teléfonos o datos sensibles).
  - Solicitar a un modelo de IA Generativa (como ChatGPT) que les genere un set de datos simulado en formato CSV basado en el problema que eligieron.
- **Planteo del problema:** Expliquen en un párrafo breve qué problema quieren resolver, quiénes son los afectados y por qué es importante tomar decisiones basadas en datos en este caso en particular.

### Paso 2: Limpieza y Exploración de Datos (Módulo 2)

Utilizando herramientas que no requieran programar (como Microsoft Excel o Google Sheets), trabajen con su tabla de la siguiente manera:

- **Limpieza (Data Cleaning):** Busquen y resuelvan datos faltantes (celdas vacías), eliminen filas duplicadas y corrijan errores evidentes (ej. textos con errores ortográficos que se repiten de forma inconsistente, formatos de fecha rotos o números guardados como texto).
- **Análisis Exploratorio (EDA):** Formulen y respondan tres (3) preguntas clave utilizando los datos limpios. (Ejemplo de preguntas: ¿Cuál es el producto que genera más ingresos? ¿En qué mes se registró la menor cantidad de visitas? ¿Qué perfil de cliente realiza compras de mayor valor?).
- **Visualización de datos:** Diseñen dos (2) gráficos claros y profesionales (pueden ser de barras, de líneas, de dispersión o circulares) que representen los hallazgos más importantes de sus preguntas. Recuerden elegir el tipo de gráfico adecuado según la información que quieren mostrar y colocarles títulos claros.

### Paso 3: Propuesta Conceptual de Machine Learning (Módulo 3)

En esta sección utilizaremos la lógica del aprendizaje automático sin entrar en la complejidad técnica de la programación. Imaginen que tienen a su disposición un sistema inteligente para potenciar su solución:

- **Identificación del enfoque:** Determinen qué tipo de problema de Machine Learning se adapta mejor a su caso:
  - **Clasificación:** Si necesitan predecir una categoría o etiqueta (ej. ¿el cliente abandonará el servicio: Sí o No?).
  - **Regresión:** Si necesitan predecir un valor numérico continuo (ej. ¿cuántos ingresos estimamos recaudar el próximo mes?).
  - **Agrupamiento (Clustering):** Si necesitan encontrar patrones ocultos para agrupar elementos similares (ej. segmentar a los usuarios en categorías de comportamiento de uso).
- **Explicación conceptual:** Expliquen con palabras sencillas:
  - ¿Qué datos históricos (características) le darían a la máquina para que aprenda?
  - ¿Cómo ayudaría esa predicción futura a que la organización o persona tome una mejor decisión?
