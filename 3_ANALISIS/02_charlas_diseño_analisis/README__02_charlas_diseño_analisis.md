## Nota sobre el nombre de esta carpeta

El nombre de esta carpeta tiene una "ñ" (`diseño`). En zips generados con
ciertas herramientas de compresión, ese carácter puede corromperse a
`dise#U00f1o` por un problema de codificación al comprimir/descomprimir. Esto
se diagnosticó y corrigió en la sesión del 10-07-2026: si en algún momento
el nombre de esta carpeta vuelve a aparecer como `02_charlas_dise#U00f1o_analisis`
en un zip nuevo, es el mismo bug reapareciendo (típicamente por comprimir
sin la opción `-X` de `zip`), no un cambio real de nombre.
